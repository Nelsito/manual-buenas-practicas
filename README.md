# 📖 Manual de Buenas Prácticas Universitarias

Este repositorio contiene un manual colaborativo diseñado para estandarizar y optimizar el trabajo en equipo en proyectos académicos.

## 👥 Autores
* Nelson Peña

## 🗂️ Índice del Repositorio
* `/documentos/`: Código fuente del manual principal en Markdown.
* `/imagenes/`: Recursos gráficos de apoyo.
* `/referencias/`: Enlaces a normativas APA y herramientas.

## 🚀 Uso y Contribución
Para usar este manual, puedes leer los archivos Markdown directamente o descargar la versión en PDF. 
Si deseas contribuir:
1. Crea una nueva rama (`git checkout -b feature/tu-mejora`).
2. Haz tus cambios y súbelos (`git push`).
3. Abre un Pull Request explicando tu aporte para que el equipo lo revise.
